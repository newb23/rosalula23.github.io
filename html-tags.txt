<h2>HTML Headings</h2>

<dl>
    <!-- heading -->
  <dt>< h1 > < h2 > < h3 > < h4 > < h5 > < h6 > </dt>
  <dd>Heading tags mark what text makes up the heading. They are numbered from 1 to 6 based on how important they are.</dd>
  <dd><h1>This is an example of h1</h1>
  		<h2>This is an example of h2</h2>
        <h3>This is an example of h3</h3>
        <h4>This is an example of h4</h4>
        <h5>This is an example of h5</h5>
        <h6>This is an example of h6</h6></dd>
        <!-- body -->
  <dt>< body ></dt>
  <dd> < body > marks which text makes up the body of the document</dd>
  <dd> <body>This is an example of body. It doesn't look much different than the list items or other text. But the < body > tag was used to define the content.</body></dd>
    <!-- unordered list -->  
        <dt>< ul></dt>
  <dd> < ul > starts an unordered list. An unordered list is marked with small black circles by default. </dd>
  <dd> After using < ul >, list items are added using < li >item< /li>.</dd>
  <dd> <ul> 
  		<li>Item 1</li>
        <li>Item 2</li>
        <li>Item 3</li></ul></dd>
        <!-- ordered list -->
  <dt> < ol > </dt> 
  	<dd> < ol > stars an ordered list. This list is marked with numbers by default</dd>
    <dd> Like with < ul >, list items are added using < li >item< /li>.</dd>
    <dd> <ol>
    	<li>Item 1</li>
        <li>Item 2</li>
        <li>Item 3</li></ol></dd>
        <!-- hyperlinks -->
 <dt> < a href > </dt>
 	<dd>< a href > allows you to insert a hyperlink.</dd>
    <dd> This looks like < a href="url">link text< /a></dd>
    <dd> <a href=https://www.w3schools.com/html/html_links.asp>Here's a better explanation</a></dd>
    <dt> < img > </dt>
    <dd> < img > allows you to insert an image. </dd>
    <dd> This looks like < img src="filepath" alt="Name" ></dd>
    <dd> You can also specify the size of the image <br> 
    	< img src="filepath" alt="name" width="X" height="X"></dd>
    <dd><img src="135-1359738_transparent-background-lego-man-png.png" alt="legoman" width="100" height="200"></dd>
    <!-- strong -->
 <dt> < strong > </dt>
 	<dd> < strong > indicates text which is very important. </dd>
    <dd> <strong>This is very important!</strong></dd>
    <dd> <b>This text is bold.</b></dd>
    <dd> These look the same, but are different for those using accessibilty aids and those poking through the code. </dd>
    <!-- space entity -->
 <dt> & nbsp; </dt>
 	<dd>This addition in a line of text inputs an extra space between words without creating a new line. </dd>
    <dd>This looks like < p > text text & nbsp; text < /p ></dd>
    <dd><p>This looks like text text &nbsp; text</p></dd>
    <dd>Compare to <br>
    		<p>This looks like text text text</p></dd>
 <dd>Other entities are added in the same way: & entityname ;.</dd>
 <dd>Euro &euro; Yen &yen; Pound &pound; etc.</dd>
